# Qt 天气预报应用

一个基于 Qt 6 开发的跨平台桌面天气预报应用，支持实时天气查询和7天天气预报，绿色免安装，开箱即用。

## ✨ 功能特性

- **实时天气**：获取当前城市的温度、天气状况、体感温度、风速、湿度、气压等详细信息
- **7天预报**：显示未来7天的最高/最低温度、天气状况和风向
- **城市切换**：点击城市按钮输入搜索，支持全国所有城市的模糊匹配和一键切换
- **自动刷新**：JWT Token 每14分钟自动刷新，无需手动干预
- **简洁实用的UI**：新手第一次做Qt界面，主打一个功能完整、清晰易用
- **跨平台**：支持 Windows 10/11，理论上可编译运行在 macOS 和 Linux 上

## 🛠️ 技术栈

- **框架**：Qt 6.9.3 (MSVC 2022 64bit)
- **语言**：C++ 17、Python 3.13
- **API**：和风天气 API (QWeather v7)
- **认证**：JWT (RS256 算法)
- **构建系统**：CMake

## 🚀 快速开始

### 环境要求

- Qt 6.9.3 或更高版本（必须包含 Qt Network 模块）
- Visual Studio 2022（MSVC 2022 64bit 工具链）
- Python 3.10+（开发环境，打包后不需要）

### 前置准备

1. **获取和风天气 API 密钥**
   - 注册并登录 [和风天气开发者平台](https://dev.qweather.com/)[官方文本超详细]
   - 创建一个新项目，获取你的 `API_KEY_ID`
  
   - ### ⚠️ 重要安全警告
**本仓库已移除 `generate_jwt.py` 脚本文件**。

你需要**自行实现JWT令牌生成逻辑**，修改位置：
- 文件：`widget.cpp`
- 函数：`void Widget::refreshJwt()`

你有两种实现方式：
1. **推荐方式**：参考和风天气官方文档，用python直接在Qt中生成令牌
2. **简单方式**：自己创建一个`generate_jwt.py`脚本，实现RS256算法的JWT生成，然后修改`refreshJwt()`函数调用这个脚本
